Smart URL Redirector

How to use URL patterns:

From URL Pattern: Use Chrome's URL filter syntax:

  • Domain matching: ||google.com (matches domain and all subdomains)
  • Exact prefix: |https://google.com/ (matches exact URL prefix)
  • Wildcards: google.com* (matches any path on domain)
  • Path preservation: https://google.com/*

To URL: The destination URL. Use * for path preservation:

  • Simple redirect: https://search.google.com
  • Path preservation: https://search.google.com/*

Examples:

  • ||google.com → https://search.google.com
  • https://google.com/* → https://search.google.com/*
